# Quoin

**Quoin** is a Smalltalk-inspired programming language built on a Rust virtual
machine, with first-class *blocks* and message passing at its core.

> 🚧 **Work in progress.** This crate currently only reserves the name `quoin`
> on crates.io — there's nothing to use yet.

Follow along at **[quoinlang.dev](https://quoinlang.dev)**.

## Why "Quoin"?

A *quoin* (pronounced "coin") is the cornerstone block that anchors a wall —
and, in letterpress printing, the wedge that locks movable *type* snugly into
place. Small pieces, locked into a working whole.
